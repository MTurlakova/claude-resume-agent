# Resume Tailor — Project Instructions

## Output
- Folder: set `OUTPUT_FOLDER` in `resume_data.py`
- No subfolders — save directly to the output folder
- File name: `{First Last} Resume - {Company} {Role}.docx`

## Formatting defaults
- Font: Garamond 11pt
- Margins: 0.75 inches
- Target length: 2 pages
